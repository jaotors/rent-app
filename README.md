## Jao Rent App


### Tech Stack
- [Next.js](https://nextjs.org/)
- [Pico.css](https://picocss.com)

### How to run

```bash
~ git clone https://github.com/jaotors/rent-app
~ cd jao-rent-app
~ pnpm install
~ pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
